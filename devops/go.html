GO

Compiling code: go build example.go
Running executable: ./example
Running code without creating an executable: go run example.go

Code Syntax:

package main - creates an executable instead of a library
import "package" - imports necessary packages

func main () {
fmt.Println("Hello World")
}


Go's Package List - https://golang.org/pkg/

Importing multiple packages:

import (
p1 "Package1" - use an alias for the package
"Package2"
)

p1.SampleFunc() instead of package1.SampleFunc()

// Single-line comment
/*
Multi-line comment
*/

go doc package - go's terminal documentation for each package
go doc package.function - for more in-depth information

fmt.Println(68 * 23) // Arithmatic in Go

const // create constant
